YueLink 安装完成 — 快速上手指南
==============================

【启动 YueLink】

  • 从 开始菜单 → YueLink 启动
  • 或者双击桌面快捷方式（如果你勾选了"创建桌面图标"）
  • 安装器已自动放行 Windows 防火墙，首次启动不会弹出
    "Windows Defender 阻止"对话框

【首次使用】

  1. 启动 YueLink，登录你的悦通账号
  2. 进入「线路」页面，等待节点列表加载完成
  3. 点击任意节点连接 — 状态栏出现绿色 ✓ 即代理生效
  4. 点击「我的 → 通用」可切换 系统代理 / TUN 模式

【小提示】

  • TUN 模式需要管理员权限。第一次切换时 Windows 会弹 UAC，
    点击 "是" 即可
  • 默认使用 系统代理 模式，不需要管理员权限，覆盖所有支持
    系统代理的应用（浏览器 / 大多数桌面客户端）
  • 想要全局接管所有流量（含游戏 / 命令行工具），请切换到
    TUN 模式
  • 配置 / 日志文件位于：
    %AppData%\YueLink\
    卸载时这两个目录会被清理

【常见问题】

  • 节点连接不上？尝试 我的 → 通用 → 切换连接模式
  • 升级后图标没刷新？任务栏右键 → 取消固定 → 重新打开 → 重新固定
  • 需要帮助？访问 https://yue.to 或仓库 issues

【关于"Windows 已保护你的电脑"提示】

  YueLink 是开源软件，目前没有购买昂贵的 EV 代码签名证书，所以
  Windows SmartScreen 在首次启动时可能会弹出"Windows 已保护你的
  电脑"蓝色对话框。这不代表病毒——只是 Microsoft 还没积累足够的
  下载量来认可这个签名。

  绕过方法：
    1. 在蓝色对话框上点击"更多信息"（不是"不运行"）
    2. 出现"仍要运行"按钮，点击即可
    3. 之后再启动不会再弹

  如果你担心安全，可以用 https://www.virustotal.com/ 上传
  YueLink-Windows-Setup.exe 验证（多家杀毒引擎扫描）。

祝你使用愉快！
