## New

- 网络变化自动恢复：Wi-Fi 切蜂窝、蜂窝切 Wi-Fi、切换到另一个 Wi-Fi、桌面唤醒后网络变化，会自动触发代理重连，不再依赖手动关开代理。覆盖 Android / iOS / macOS / Windows / Linux。
- 桌面 Service Mode 安装入口：设置页和连接修复页都能直接安装 / 更新，首次使用不再因状态刷新失败而看不到按钮。

## Fixes

- Android Wi-Fi 与蜂窝网络互切后偶发断流。
- iOS / macOS 通过 NWPathMonitor 主动通知 Flutter 层恢复代理。
- macOS 网络变化后 TUN DNS 偶尔未重新应用，停止代理后无法还原原始 DNS。
- mihomo 本地控制接口（REST / WebSocket）在系统代理开启时被绕回 mixed-port 导致 5xx。
- 网络切换后失败重试改为分阶段，仍失败时执行轻量重启，减少需要手动重开的情况。

## Upgrade Notes

- 已登录用户无需重登，订阅与设置自动保留。
- 直接覆盖安装即可，所有平台。
- Android 用户建议重点验证 Wi-Fi 与蜂窝互切。
- macOS：DMG 直接覆盖；TUN 模式首次使用可在设置或连接修复页安装 Service Mode。
