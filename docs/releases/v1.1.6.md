## Fixes

- Windows 已安装服务模式后开启 TUN，即使路由、DNS、连通性全部正常，仍会被误判为"需要管理员权限"并弹红色横幅。判定逻辑改为检查特权 helper 是否注册（与 macOS / Linux 一致），不再看 UI 进程是否提权。

## Upgrade Notes

- 已登录用户无需重登，订阅与设置自动保留。
- 默认匿名遥测保持关闭。开启位置：设置 → 隐私 → 匿名遥测。
- 直接覆盖安装即可，所有平台。
