# YueLink v1.0.24

正式版，2026-04-29。本轮聚焦两件事：**iOS 巨魔（TrollStore）连接失败诊断**，以及 **签到日历 + 补签卡的全 native UI**。预发布 v1.0.24-pre 同日发出，真机三平台验证通过。

## What's New

- **签到日历 native 月视图** — 设置 → 订阅相关 → 签到日历 / Dashboard 独立卡片。emoji 网格表达签到状态（✅ 已签 / ⭐ 补签 / ⛔ 断签 / 🔴 今天未签 / ⏳ 未来），头部摘要含连签天数 / 本月已签 / 距下个里程碑（7/14/30/60/90 天）。支持月份左右切换 + 下拉刷新。
- **补签卡 native UI** — 断签当天日历底部出现「⭐ 用 25 积分补昨天」按钮，弹窗显示当前积分余额 + 后端权威价格，扣积分 + 后端写历史 + 维持连签 streak 一气呵成。
- **CheckinCard 已签副标题显示连签** — 签到完仪表盘卡片副标题显示「已签到 · 连签 N 天」（仅 ≥2 天显示，避免新用户首签 1 天啰嗦）。
- **Dashboard 加签到日历独立入口卡** — 与签到 / Emby 平级，未签到时也显眼可达，承载月度成就感。
- **CheckinCard 加图标按钮直跳日历** — 已签到 / 已在其他设备签到时显示日历图标，一键直跳月历。

## iOS 巨魔（TrollStore）修复

历史用户报告："点连接，不弹添加 VPN 提示，提示连接成功但实际无网络"。这一版正面解决。

- **fail-loud 检测** — 连接成功后若 10 秒内被系统中止，App 自动判定为 entitlement 可疑，弹出全屏「iOS 安装方式」引导页，红框警告 + AltStore / SideStore 推荐。
- **iOS 安装方式说明页** — 设置 → 关于 → iOS 安装方式 主动入口；列出三种方式（AltStore / TrollStore / IPA 直装）的 pros / cons / 操作步骤，明确告知巨魔不支持 VPN（NetworkExtension 仍依赖 Apple 签发的 provisioning profile，巨魔 IPA 拿不到这条信任链）。
- **README iOS 章节** — 加红框警告，明确「巨魔可装但 VPN 不工作」+ 推荐 AltStore / SideStore。

## 国际化

- **签到日历 + 补签卡 + iOS 安装方式 全 i18n** — 56 个新 key 加到 zh / en。中英都已校对，英文为产品级文案（非机翻），周一-周日缩写按 iOS HIG 用 3 字母（Mon / Tue / ... / Sun）。
- 切换语言：设置 → 语言 → English / 中文。

## API 协议变更（与 yueops 后端协同）

- 新增 `GET /api/client/checkin/history?month=YYYY-MM` — 月历数据，把 `gambling_points` 和 `sign_card_cost` 一并发回，省一次往返。
- 新增 `POST /api/client/checkin/resign` — 25 积分补昨天 + 维持 streak。
- 新建 `app_sign_history` 表（独立于 Telegram bot 通道），保留双签政策。

## Bug Fixes

- **iOS 巨魔下「提示连接成功但实际无网络」** — 现在会明确告知是 entitlement 信任链问题，引导改用 AltStore / SideStore（见上）。
- **`CheckinResult.streak` 字段** — 之前签到结果 model 里没解析 streak，导致依赖此字段的 UI 无从读起。补全后 CheckinCard 能稳定渲染连签天数。

## Upgrade Notes

- 已登录用户无需重登，订阅与设置自动保留。
- iOS 巨魔用户：建议改用 AltStore / SideStore 自签后重装 IPA，VPN 才能真正工作。已经能连接的用户继续使用即可，本次不破坏现有安装。
- AltStore / SideStore / TrollStore 重装新 IPA 即可。
- macOS / Windows / Linux：直接覆盖。
- Android：直接安装新版 APK。
- **用过 1.0.24-pre 的**：与正式版无功能差异，可以直接覆盖。
