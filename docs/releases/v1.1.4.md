## New

- 轻玻璃视觉系统：底部 tab、主页面背景、列表区块、设置卡片、订阅卡、套餐卡、节点组、签到卡、公告卡统一为半透明玻璃表面。滚动卡片避免重 blur，不再因为视觉效果牺牲滚动流畅度。

## Fixes

- 中文 Windows 下 `netsh` / `ipconfig` 输出乱码。
- 日志导出遇到非 UTF-8 字节时直接失败，现在尽量保留可读内容。
- 连接修复页继续区分控制接口、TUN 网卡、路由、DNS、系统代理冲突、节点 timeout、AI 出口受限和 Reality 认证失败，不再把所有问题都写成"节点不可用"。

## Changed

- 二级页面统一为紧凑分组结构，桌面端自动限制内容宽度。
- Windows 构建流程下载并校验 Wintun，构建后检查 `wintun.dll` 是否进入产物，缺失时阻断发布。

## Upgrade Notes

- 已登录用户无需重登，订阅与设置自动保留。
- 默认匿名遥测保持关闭。开启位置：设置 → 隐私 → 匿名遥测。
- 直接覆盖安装即可，所有平台。
- 打开代理后仍显示"连接失败"，请进入 设置 → 连接修复 查看具体归因再导出诊断包。
- Windows 仍提示"Wintun 驱动缺失"请使用正式安装包重装。
