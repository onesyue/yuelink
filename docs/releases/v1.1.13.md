## Fixes

- 悦视频倍速选择区分"用户选择的常规倍速"与"长按临时 2× 加速"，点击屏幕或手势取消不再把 1.25× / 1.5× / 2× 清回 1×。
- 速度面板勾选跟随用户实际选择的倍速，顶部倍速标签显示为 `2x` 而非 `2.0x`。
- 中文用户名或中文临时路径下 Windows 特权服务安装失败：服务安装、更新、卸载脚本改为 UTF-8 BOM 写入并通过 EncodedCommand 启动。

## Upgrade Notes

- 这是 v1.1.12 的 hotfix。已登录用户无需重登，订阅、设置和播放记录自动保留。
- Windows 升级后仍显示"特权服务未安装"，请进入 设置 → 连接修复 → 桌面 TUN 重新点击"安装服务"。
