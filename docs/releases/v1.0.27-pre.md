# YueLink v1.0.27-pre

预发布版，2026-04-29。本轮聚焦 **iOS 26 风全 UI 精细化** —— 新建 design system 组件 + 6 个高频页迁移到大标题 scaffold + 我的 tab 全面重构成 inset grouped list。

## What's New

- **iOS 26 大标题滚动收起** — 我的 / 订阅套餐 / 订单记录 / 订阅管理 / 公告 / 意见反馈 / iOS 安装方式，7 个页面统一用 `YLLargeTitleScaffold`：进入页面是 32pt 大标题，向上滚动自动 collapse 到 18pt 小标题，跟 iOS 26 Settings / Apple Music / 通讯录一致的体感。
- **我的 tab 全面重构** — 三组 inset grouped 卡片（服务 / 设置 / 关于），每行标准 anatomy（彩色 squircle 图标 + 16pt 行标题 + 标准 trailing：chevron / value / 状态徽章 / 加载圈）。section header 大写 caps 11pt + letter-spacing 0.6，整体观感对齐 Apple 通讯录的精致质感。
- **签到日历重画 + 玻璃底栏** — 继承 v1.0.26 内容（详见 v1.0.26 release notes）。
- **9 个 settings 子页一键升级** — 偏好 / 覆写 / 修复 / DNS 查询 / 代理 provider / 运行配置 / 遥测预览 / 启动报告 / 站内 web 页面，所有这些子页通过升级底层 `SettingsCard / YLInfoRow` 视觉自动对齐新 design system，**调用站零改动**。零回归风险。
- **意见反馈页 redesign** — 表单字段 rounded surface + 大标题 scaffold + 统一 FilledButton 提交按钮。
- **iOS 安装方式说明页 redesign** — 三种侧载方式卡片去掉 border、tag 改 pill、howto 加 line-height、统一 spacing。

## Bug Fixes

- 继承 v1.0.26 build 127 的 Android 启动 ANR 修复（玻璃底栏 BackdropFilter sigma 平台分流）和切换节点 UI 卡顿修复。

## Upgrade Notes

- 已登录用户无需重登，订阅与设置自动保留。
- AltStore / SideStore 用户：重装新 IPA 即可。
- macOS / Windows / Linux：直接覆盖。
- Android：直接安装新版 APK。
- 用过 v1.0.26 的：与正式版无功能差异，纯 UI polish。
