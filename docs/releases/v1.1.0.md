# YueLink v1.1.0

正式版，2026-04-30。本轮主线把 Android 切 tab 的 ANR、剩余页面的 UI 风格、底部菜单栏的连接状态可视化都对齐了 iOS 26 标准。

## What's New

- **设计系统覆盖剩余 12 个 page** — 设置子页（DNS 查询 / 通用偏好 / 配置覆写 / Provider / 启动报告 / 连接修复 / 运行配置 / 遥测预览）、Surge 模块（列表 / 详情 / 证书引导）和签到日历都迁到 `YLLargeTitleScaffold` + `YLSection / YLListTile`，跟首页 / 线路 / 我的同一套大标题、卡片留白、间距。整 app 视觉对齐到 iOS 26 / 通讯录密度。
- **图标统一为 rounded 实心风格** — 全 app 48 种 Material 图标从 outlined 细线轮廓换成 rounded 实心，对齐 iOS 26 / SF Symbols `.fill` 视觉权重，小尺寸下不再糊、不再混搭。
- **macOS 菜单栏图标按状态变色** — 未连接深色、已连接浅色，扫一眼菜单栏就知道当前是否在保护中。Windows 系统托盘保留彩色 .ico 不变。
- **节点匿名指纹 v2** — 订阅解析后上报 `node_inventory.nodes`，每个节点带 `fp / type / xb_server_id`，缺数字 ID 时回退 `sid`。`node_urltest` 用同一份 fp 缓存，测速结果可与 inventory 精确关联。
- **iOS 26 风全 UI 精细化** — 首页 / 线路 / 悦视频 / 我的移除顶部大字标题，留白、字体、列表、底部 tab 贴近 Apple 通讯录 / 电话 app 的密度和层级。
- **系统字体栈** — 桌面与移动端改用平台系统字体，移除 Google Fonts 依赖，降低启动和离线字体风险。
- **悦视频代理感知** — 媒体库和图片请求只在核心运行时走本地代理，未连接时不再请求 `127.0.0.1` 导致 connection refused。

## Bug Fixes

- **Android 切 tab 不再卡死** — v1.0.26 引入的玻璃底部导航在 Adreno GPU 上单帧成本爆掉，10 秒主线程冻结直接 ANR。回退 CupertinoTabBar，首次切到未建过的 tab 也延后一帧 build，输入永不堆积。已在 SM-G9860 / Android 13 验证 5 分钟 21 次切 tab 0 ANR。
- **Android 启动卡死 / 无响应** — 移除冷启动时的 eager FlutterEngine 预热，改为按需缓存，降低 Android 原生启动阶段阻塞。
- **底部 tab 遮挡页面** — 底部导航和页面内容参与同一布局,避免盖住列表末尾。
- **悦视频错误提示** — 本地代理未启动导致的 127.0.0.1 refused 会显示连接提示，不再误判为 UI 卡死。
- **测速失败事件补 reason** — 失败测速标准化为 `delay_ms=5000` + `reason=timeout`，同时携带节点身份字段。
- **telemetry sanitizer 修复** — `List<Map<String, scalar>>` 不再被清空，`node_inventory.nodes` 能真实送到服务端。
- **Feature flags 启动顺序** — flags 请求等匿名 `client_id` 初始化完成后再发，空 client_id 会被跳过。

## Guardrails

- Android profile 模式手机实测：Samsung Galaxy S21 FE (SM-G9860) / Android 13 / Adreno 660，5 分钟 21 次切 tab 全部正常，logcat 无 ANR / SIGQUIT。
- 整个项目 728 个测试通过，覆盖 v2 fp 合约、`sid` hash fallback、WebSocket Host fallback、inventory 不泄露 server/port/password/sni、urltest 与 inventory fp 一致、失败 reason、flags URI 等回归。

## Upgrade Notes

- 已登录用户无需重登,订阅与设置自动保留。
- Android：直接安装新版 APK。如果之前是 v1.0.26 / v1.0.27 / v1.1.0-pre 受切 tab ANR 困扰，强烈建议升级。
- iOS：AltStore / SideStore 用户重装新 IPA 即可。
- macOS / Windows / Linux：直接覆盖。
- 用过 v1.1.0-pre 的：与正式版功能一致，差异是新增切 tab ANR 修复、更多设置子页迁到设计系统、图标统一 rounded 实心、macOS 菜单栏图标变色。
