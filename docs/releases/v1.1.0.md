## What's New

- **Android 切换 tab 不再卡死** — v1.0.26 的玻璃底部导航在 Adreno GPU 上单帧成本爆掉，10 秒主线程冻结直接 ANR；现在退回 CupertinoTabBar，首次切到未建过的 tab 也延后一帧 build，输入永不堆积。在 Samsung Galaxy S21 FE / Android 13 上实测 5 分钟 21 次切 tab 0 ANR。
- **设计系统覆盖剩余 12 个页面** — DNS 查询 / 通用偏好 / 配置覆写 / Provider / 启动报告 / 连接修复 / 运行配置 / 遥测预览 / Surge 模块（列表 / 详情 / 证书引导）/ 签到日历都迁到 `YLLargeTitleScaffold` + `YLSection / YLListTile`，跟首页 / 线路 / 我的同一套大标题、卡片留白、间距，整 app 视觉对齐到 iOS 26 / 通讯录密度。
- **图标统一为 rounded 实心风格** — 全 app 48 种 Material 图标从 outlined 细线轮廓换成 rounded 实心，对齐 iOS 26 / SF Symbols `.fill` 视觉权重，小尺寸下不再糊、不再混搭。
- **macOS 菜单栏图标按状态变色** — 未连接深色、已连接浅色，扫一眼菜单栏就知道当前是否在保护中；Windows 系统托盘保留彩色图标不变。
- **节点匿名指纹 v2** — 订阅解析后上报的 `node_inventory.nodes` 每个节点带 `fp / type / xb_server_id`，缺数字 ID 时回退 `sid`；测速结果可与 inventory 精确关联。指纹算法本地 SHA-256，原始 server / port / password / sni 永远不出本机。
- **iOS 26 风全 UI 精细化** — 首页 / 线路 / 悦视频 / 我的去掉顶部大字标题，留白、字体、列表、底部 tab 贴近 Apple 通讯录 / 电话 app 的密度和层级。
- **系统字体栈** — 桌面与移动端改用平台系统字体（PingFang / Noto Sans CJK / Microsoft YaHei UI），移除 Google Fonts 在线依赖，启动更稳、离线场景更可靠。
- **悦视频代理感知** — 媒体库和图片请求只在核心运行时走本地代理，未连接时不再请求 `127.0.0.1` 导致 connection refused；提示卡引导用户先开启保护。

## Bug Fixes

- **Android 启动卡死 / 无响应** — 移除冷启动时的 eager FlutterEngine 预热，改为按需缓存，降低 Android 原生启动阶段阻塞。
- **底部 tab 遮挡页面** — 底部导航和页面内容参与同一布局，避免盖住列表末尾。
- **测速失败事件补 reason** — 失败测速标准化为 `delay_ms=5000` + `reason=timeout`，同时携带节点身份字段。
- **Telemetry sanitizer** — `List<Map<String, scalar>>` 不再被清空，`node_inventory.nodes` 能真实送到服务端。
- **Feature flags 启动顺序** — flags 请求等匿名 `client_id` 初始化完成后再发，空 client_id 会被跳过，避免一次性的失败上报。

## Upgrade Notes

- 已登录用户无需重登，订阅与设置自动保留。
- Android：直接安装新版 APK；之前在 v1.0.26 / v1.0.27 / v1.1.0-pre 上遇到切 tab ANR 的，强烈建议升级。
- macOS：DMG 直接覆盖；菜单栏图标在连接前后会自动换色。
- Windows：直接安装新版，菜单栏图标保持彩色不变。
- Linux：AppImage 直接覆盖。
- iOS：TrollStore / AltStore / SideStore 重装新 IPA 即可。
- 用过 1.1.0-pre 的：差异是新增切 tab ANR 修复、更多设置子页迁到设计系统、图标统一 rounded 实心、macOS 菜单栏图标变色，正式版可以直接覆盖。
