## What's New

- **Android 切换 tab 不再卡死** — 一些机型上从首页切到节点 / 悦视频 / 我的会冻几秒甚至弹"程序无响应"对话框，已彻底修复。底部导航和页面渲染整体重做，再频繁来回切也不会丢响应。
- **设计系统覆盖剩余页面** — DNS 查询 / 通用偏好 / 配置覆写 / Provider / 启动报告 / 连接修复 / 运行配置 / 遥测预览 / Surge 模块（列表 / 详情 / 证书引导）/ 签到日历整组迁到统一的大标题样式，跟首页 / 线路 / 我的同一套留白、卡片、间距，整 app 视觉到此完全对齐。
- **图标统一为实心圆润风格** — 全 app 几十种图标从细线轮廓换成实心圆润，对齐 iOS 26 风格，列表里的小图标不再糊、不再粗细混搭。
- **macOS 菜单栏图标按状态变色** — 未连接深色，已连接浅色，一眼就能看出当前是否在保护中。Windows 系统托盘图标保持彩色不变。
- **节点匿名指纹** — 订阅同步后，节点会以匿名指纹形式记录，方便测速结果与节点对应起来用于后续优化。原始服务器地址、端口、密码不会出本机。
- **iOS 26 风全 UI 精细化** — 首页 / 线路 / 悦视频 / 我的去掉顶部大字标题，留白、字体、列表密度更接近 Apple 通讯录 / 电话 app。
- **系统字体栈** — 桌面与移动端改用平台系统字体（PingFang / 思源黑体 / 微软雅黑），不再依赖在线字体下载，启动更稳、离线更可靠。
- **悦视频代理感知** — 媒体库和图片请求只在已连接时走代理，未连接时不再无谓地请求本地端口报错；提示卡引导用户先开启保护。

## Bug Fixes

- **冷启动 / 启动后无响应** — 启动阶段流程精简，更不容易因为系统繁忙拖到无响应。
- **底部导航遮挡列表末尾** — 底部导航和页面内容参与同一布局，最后一项不会被盖住。
- **测速失败补充原因** — 失败的测速会标明是超时还是其他原因，便于定位"为什么这条线连不上"。
- **遥测发送遗漏** — 节点清单上报修复，统计数据更完整。
- **Feature flags 取值时机** — 等用户匿名标识就绪再发请求，避免一次性的失败上报。

## Upgrade Notes

- 已登录用户无需重登，订阅与设置自动保留。
- Android：直接安装新版 APK；之前在 v1.0.26 / v1.0.27 / v1.1.0-pre 上遇到切 tab 卡死的强烈建议升级。
- macOS：DMG 直接覆盖；菜单栏图标在连接前后会自动换色。
- Windows：直接安装新版，菜单栏图标保持彩色不变。
- Linux：AppImage 直接覆盖。
- iOS：TrollStore / AltStore / SideStore 重装新 IPA 即可。
- 用过 1.1.0-pre 的：差异是新增切 tab 卡死修复、剩余设置子页迁到设计系统、图标统一实心圆润、macOS 菜单栏图标变色，可以直接覆盖。
