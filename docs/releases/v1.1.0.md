## New

- 节点匿名指纹：测速结果与节点一一对应，原始服务器地址、端口、密码不出本机。
- macOS 菜单栏图标随连接状态变色，未连接深色、已连接浅色。
- 系统字体栈：桌面与移动端统一使用平台系统字体（PingFang / 思源黑体 / 微软雅黑），启动更稳、离线可靠。

## Fixes

- Android 部分机型切换底部 tab 卡顿、弹"程序无响应"对话框。
- 冷启动后短暂无响应。
- 底部导航遮挡列表最后一行。
- 测速失败补充失败原因（超时 / 网络不可达 / TLS 错误）。
- 节点清单遥测漏发。

## Changed

- 12 个二级页面（DNS 查询、配置覆写、连接修复、运行配置、Provider、启动报告、签到日历等）切到统一大标题样式，与首页 / 线路 / 我的留白对齐。
- 全局图标改实心圆润风格，列表小图标更清晰。
- 首页 / 线路 / 悦视频 / 我的去掉顶部大字标题，留白接近 Apple 通讯录 / 电话。
- 悦视频未连接时不再发请求，提示先开启保护。

## Upgrade Notes

- 已登录用户无需重登，订阅与设置自动保留。
- 直接覆盖安装即可，所有平台。
- 此前在 v1.0.26 / v1.0.27 / v1.1.0-pre 上遇到切 tab 卡死的强烈建议升级。
