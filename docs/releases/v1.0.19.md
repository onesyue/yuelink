## What's New

- **QUIC 策略可配置**(`off` / `googlevideo` / `all`,默认 `googlevideo`)。原本硬编码的全局 UDP:443 REJECT-DROP 改为用户可切换。
  - `googlevideo`(默认):只对 YouTube 视频 CDN 做 TCP 回落。
  - `off`:不拦截 QUIC,适配 MyTV Super 等依赖 HTTP/3 的港区流媒体。
  - `all`:保留全局 UDP:443 拒绝,仅用于排障。
  - UI 位置:**设置 → 连接 → QUIC 策略**(中英双语)。
- **TUN 模式靛蓝高亮** `#6366F1`:运行中的 TUN 模式,Dashboard 连接卡边框 / 状态点 / 「已连接」/ 电源按钮 / 路由模式徽章 / 模式徽章 同时从翠绿切到靛蓝,一眼区分跑的是哪种模式。系统代理保持翠绿 `#10B981`。
- **托盘菜单顶部状态项可点击打开主窗口**:「YueLink · 已连接 · 节点名」之前是不可点击的 label,现在点一下直接拉起主窗口。macOS 左键 tray 仍弹菜单;Windows 左键 tray 仍打开窗口。

## Improvements

- **experimental 回归 mihomo 上游默认**:不再默认注入 `quic-go-disable-gso` / `quic-go-disable-ecn`,这两项在非病态网络上每项约减 hy2 / QUIC 吞吐 20–30%。订阅显式配置保持不动。
- **TUN MTU 统一到 1500**:桌面冷启动 / Android / iOS fd TUN / 热切换 PATCH 都走同一常量 `AppConstants.defaultTunMtu`,消除旧版「冷启动 1500 / 热切换 9000」不一致。

## Bug Fixes

- Build 工作流对 `pre` / `*-pre` tag 的 prerelease 判定修正。
