## New

- iOS 26 风格底部导航：玻璃材质 + 切换图标动画。Android 端同步规避带玻璃模糊的卡顿路径，切 tab 不再卡顿。
- macOS 任务栏图标随连接状态切色，无需打开主窗口即可确认状态。
- Windows 安装包内置 Wintun 驱动：从上游下载并校验 sha256 后再打包，缺驱动时界面明示"Wintun 驱动缺失"，不再误报"已连接"。

## Changed

- 12 个二级页面（订阅、套餐、订单、签到、偏好、配置覆写、连接修复、规则模块、公告等）统一为紧凑分组样式，去掉重复大标题；桌面端自动收紧最大宽度，不再撑满整屏。
- 桌面 TUN 状态判定要求控制接口、TUN 网卡、路由、DNS、基础连通性五项全过才显示"已连接"，缺一项给出具体原因。

## Backend

- 节点状态机基础设施上线：根据真实使用数据 + 主动探测打健康标签，能识别"普通站正常但 ChatGPT 403"等具体场景。任何下线动作都需要人工确认，不会自动把节点踢出默认组。
- 凭据保护：CI 上线敏感字段扫描器（DSN、订阅 token、Reality 公钥），含敏感字段的发布会被阻断。

## Upgrade Notes

- 已登录用户无需重登，订阅与设置自动保留。
- 默认匿名遥测保持关闭。开启位置：设置 → 隐私 → 匿名遥测。
- 直接覆盖安装即可，所有平台。
- Windows 仍提示"Wintun 驱动缺失"请到 设置 → 连接修复 导出诊断包反馈。
