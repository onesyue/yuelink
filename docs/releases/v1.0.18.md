### 发布亮点

- **视频卡顿修复（QUIC 回落）**：客户端配置模板强制 reject 所有 UDP:443 流量，Chrome/YouTube/Cloudflare/TikTok 等 QUIC 站点自动回落 TCP/TLS，彻底解决「封面加载很快、视频播放很卡」。与面板订阅模板 + 节点审计规则三层对齐，任意渠道订阅都能享受一致体验。
- **XBoard API 三主机轮询**：CloudFront 主站 502/503 或 TLS 握手失败时，自动尝试备用域名与直连源站，登录 / 用户信息 / 签到可用性显著提升，缓解「用户名一直加载中」类的偶发故障。
- **Android 稳定性加固**：连接状态机与面板故障转移逻辑收紧，减少 VPN 异常卡死与 panel 不可达时的悬挂。
- **CI 升级**：`setup-android` 升级至 v4，支持 Node.js 24 runner，应对 GitHub Actions 2026 年环境强制升级。

### 校验说明

- 每个正式产物都会附带一个同名 `.sha256` 文件，用于校验下载完整性。
- Android 用户优先下载 `android-universal.apk`；只有明确知道设备架构时，才建议改下分 ABI 包。
- Windows 默认推荐 `setup.exe`；只有不方便安装或需要绿色版时，再使用 `portable.zip`。
- iOS 包为侧载用途，不走 App Store 分发。

### 升级提示

- **全量自动生效**：QUIC reject 规则通过客户端配置模板注入，升级后首次启动即可享受；同时面板订阅刷新也会带上相同规则（三层防御），无需手动操作。
- **已登录用户**无需重登，订阅与设置自动保留。
- **非 443 的 UDP 流量**（游戏、VoIP、WebRTC、DNS）**不受影响**，仅 UDP:443（QUIC）被定向 reject 以强制 TCP 回落。
