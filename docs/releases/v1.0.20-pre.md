## What's New

- **主页模式徽章可点击切换**:「规则 / 全局 / 直连」点一下轮转切换;「TUN / 系统代理」点一下热切换(桌面)。

## Bug Fixes

- **Windows 托盘退出卡死兜底**:3 秒硬兜底 `Process.killPid`,不再出现「窗口关了但任务管理器里还在」。
- **前后台恢复收紧**:尊重用户手动停止;resume 处理器串行化;WebSocket 空闲看门狗;运营商启动去重。
- **代理保护**:三方代理劫持 30 秒内响应恢复;Linux 不可验证场景跳过破坏性 tamper restore。
- **测速**:单组全超时后,重连成功自动清空超时缓存。
- **Riverpod 3 生命周期**:post-dispose state write 在 5 个 notifier 上全部守护。
- **runtime 诊断**:4 处 silent catch 按语义分流到 `ErrorLogger` / `EventLog`。
- **更新器**:legacy check 失败只在手动检查时报给用户。
- **geodata**:CI SHA-256 严校验;运行时镜像 fail-soft。
- **构建工具链**:Android AGP 8.9.1 / Gradle 8.11.1;iOS 部署目标对齐;mihomo v1.19.24 `GOFLAGS=-mod=mod`。
