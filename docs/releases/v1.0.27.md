## What's New

- **iOS 26 风全 UI 精细化** — 我的 / 订阅套餐 / 订单记录 / 订阅管理 / 公告 / 意见反馈 / iOS 安装方式，7 个页面引入 design system 重构。基础组件 `YLSection` (inset grouped 卡片，0.33pt 内分隔线，左缩进对齐文字列) + `YLListTile` (统一行 anatomy：彩色 squircle 图标 + 14pt 行标题 + 标准 trailing) 让所有列表行视觉一致，对齐 Apple 通讯录 / iOS 26 Settings。
- **9 个 settings 子页一键升级** — 偏好 / 覆写 / 修复 / DNS 查询 / 代理 provider / 运行配置 / 遥测预览 / 启动报告 / 站内 web 页，通过升级底层 `SettingsCard / YLInfoRow` 视觉自动获得新设计，调用站零改动，零回归风险。
- **我的 / 悦视频顶部去掉大字标题** — 跟首页（仪表盘）一样直接进内容，bottom-nav 已经标识当前 tab。配合字号统一到 14pt（与首页 body 一致），整体观感对齐。
- **桌面 max-width 720dp 居中** — 所有用大标题 scaffold 的页面在桌面屏宽（mac/win/linux）自动居中到 720dp 内，对齐 Apple 桌面 System Settings / Music / Slack / VS Code / TG Desktop 惯例。不再像 v1.0.26 那样把列表撑满 1690px 内容区。
- **意见反馈页 + iOS 安装方式说明 redesign** — 表单字段 rounded surface + 大标题 scaffold；安装方式三种侧载方式卡片去掉 border、tag 改 pill、howto line-height 1.5。

## Bug Fixes

- 继承 v1.0.26 build 127 的 **Android 启动 ANR 修复** — 玻璃底栏 BackdropFilter sigma 平台分流（iOS/macOS/Windows/Linux 24，Android 8 / O(σ²) 14× 减负）。中低端 Android 设备启动不再卡死。
- 继承 v1.0.26 build 127 的 **切换节点 UI 卡顿修复** — PUT 204 后乐观更新本地选中态，refresh 改后台异步对账，spinner 在 PUT 异常时也保证复位。
- 继承 v1.0.26 的 **签到日历 emoji 状态在三星 / 小米 / 苹果机型渲染不一致** — 改用色彩弱底纹 + 角标小图标统一渲染。
- 继承 v1.0.26 的 **Android Quick Settings 磁贴图标像手铐** — 按 launcher logo 比例重画双环 + cherry-stem V。

## Upgrade Notes

- 已登录用户无需重登，订阅与设置自动保留。
- AltStore / SideStore 用户：重装新 IPA 即可。
- macOS / Windows / Linux：直接覆盖。
- Android：直接安装新版 APK。
- 用过 1.0.27-pre / 1.0.26 build 127 的：与正式版无功能差异，纯 UI polish 累积。
- 用过 1.0.26 build 126（已下架）的：必须升级，否则 Android 启动 ANR / 切换节点 UI 不更新。
