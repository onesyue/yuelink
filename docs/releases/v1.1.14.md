## Fixes

- **登录不再被订阅同步拖超时** — 登录拿到有效 token 后立即进入应用，`/getSubscribe` 与订阅配置同步改为后台执行；bootstrap 请求会在主 host 慢响应时启动备用 host 竞速，减少 DNS/CDN 抖动造成的“登录超时”体感。
- **登录 / 刷新异步状态防串号** — auth 状态机新增 session generation guard，旧登录、旧刷新、旧订阅同步返回后不能覆盖新会话，也不能用旧 401/403 把新会话踢下线。
- **“我的信息”不再经常空白** — 账户卡片优先使用本地缓存的 XBoard profile 渲染套餐、流量、到期和设备信息；独立 overview 服务不可用时保留可用信息，并在后台刷新。
- **偏好设置二级页布局修复** — 设置、诊断、模块详情等二级页面统一收窄内容宽度；复杂右侧控件在窄宽度下自动下沉，避免左右布局割裂、长文案挤压和控件溢出。
- **列表右侧值宽度修复** — 设置列表的长版本号、状态值、下拉控件等右侧内容被约束在合理宽度内，不再把左侧标题压到异常换行。
