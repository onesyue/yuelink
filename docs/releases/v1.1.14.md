## Fixes

- 登录被订阅同步拖超时：拿到有效 token 后立即进入应用，订阅同步与配置下载改为后台执行；主 host 慢响应时启动备用 host 竞速，减少 DNS / CDN 抖动造成的"登录超时"。
- 旧登录、旧刷新、旧订阅同步返回不能覆盖新会话，也不能用旧的 401/403 把新会话踢下线。
- "我的"页常出现空白：账户卡片优先用本地缓存的 XBoard profile 渲染套餐、流量、到期、设备信息；独立 overview 服务不可用时保留可用信息并在后台刷新。
- 设置页二级页面长版本号、状态值、下拉控件等右侧内容被压缩进合理宽度，不再把左侧标题挤换行；复杂右侧控件在窄宽度下自动下沉。

## Upgrade Notes

- 已登录用户无需重登。登录态、订阅、Emby、播放记录、本地设置全部保留。
- 这版主要修复登录可靠性、账户卡片展示和设置页布局，不涉及 mihomo 内核升级。
