### 发布亮点

- 应用内更新升级为稳定版 / 预发布双通道，支持多源 manifest 拉取与 SHA-256 校验。
- 桌面端正式补齐完整发布形态：Windows 安装包与便携包、macOS 通用 DMG、Linux AppImage。
- 发布链路与桌面安装体验一起打磨：全平台 squircle 图标、Windows 安装向导图、macOS DMG 背景与修复脚本。
- 核心架构完成一轮大整理：统一 `ClashCore` 接口、拆分 `CoreManager`、XBoard API 模块化、国际化迁移到 slang。
