## New

- iOS 专用 TUN 诊断面板：连接修复页针对 iOS 仅展示 Core / DNS / 目标站三层可操作状态，不再套用桌面端的 driver / 路由 / admin 模型。

## Changed

- 套餐卡片"续订 / 立即订阅"按钮固定贴齐内容区右侧，价格保持左侧对齐，避免按钮在宽屏或长价格文案下漂到中间。

## Upgrade Notes

- 已登录用户无需重登，订阅与设置自动保留。
- 默认匿名遥测保持关闭。开启位置：设置 → 隐私 → 匿名遥测。
- 直接覆盖安装即可，所有平台。
