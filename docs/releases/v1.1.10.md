## Improvements

- **订阅套餐按钮右对齐** — 套餐卡片里的「续订 / 立即订阅」按钮现在固定贴齐卡片内容区右侧，价格仍保持左侧对齐，避免按钮在宽屏或长价格文本下偏到中间。
- **iOS TUN 分层诊断** — 连接修复页新增 iOS 专用 TUN 诊断面板，聚焦 iOS 用户真正可操作的 Core、DNS、目标站三层状态，不再套用桌面端 driver / route / admin 诊断模型。
- **Riverpod 3 迁移第一批** — `xboard_api`、deep link、service mode refresh、connections snapshot/search、delay test 状态从 legacy `StateProvider` 迁到 `NotifierProvider`，为后续 Riverpod 升级减少风险。
- **关键 UI 回归保护** — 新增 Mine 流量卡、套餐按钮对齐、页面玻璃背景、`TextAlign.end`、`YLRadius` token 使用等回归测试，覆盖最近几轮视觉收口容易回退的位置。
