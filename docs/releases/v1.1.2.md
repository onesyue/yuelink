## What's improved

- **界面密度修复** — 收紧订阅、套餐、订单、签到、偏好设置、配置覆写、连接修复、规则模块和公告等二级页面的标题、列表、卡片和按钮密度。页面更接近高频工具，不再像营销页。
- **节点测速遥测** — 开启匿名遥测的用户，App 在自动测速时会上报每个节点的响应数据（成功标志 / 延迟 / 错误类型 / 状态码）。客服侧能从真实使用数据定位慢节点，不再需要让你手动复现。
- **桌面 TUN 诊断增强** — 桌面 TUN 不再只因 core 启动就显示已连接；会检查控制接口、TUN 网卡、路由、DNS 和基础连通性。异常时会显示具体原因，例如驱动缺失、权限不足、路由未接管或 DNS 接管失败。
- **目标站点归因增强** — Claude / ChatGPT 的 403、timeout、TLS EOF 会和普通节点 timeout 分开记录，帮助后续治理 AI 出口池。
- **隐私字段白名单** — 节点遥测字段被白名单锁死：节点 fingerprint 哈希、协议、目标站点、成功标志、延迟、错误类型、状态码、客户端版本、连接模式。**服务器地址 / 端口 / UUID / 密码 / SNI / Reality public-key / short-id 永不离开本机**，单元测试硬断言保护。

## Upgrade Notes

- 已登录用户无需重登，订阅与设置自动保留。
- 默认遥测保持关闭。开启位置：设置 → 隐私 → 匿名遥测。
- 直接覆盖安装即可，所有平台。
- 本版本是 UI 修复 + 遥测/诊断增强版本。节点池稳定性、AI 出口池和桌面 TUN 全平台长周期稳定性仍在治理中，不应解读为节点、AI 或 TUN 的稳定性治理已经完成。
