# YueLink v1.0.26-pre

预发布版，2026-04-29。本轮聚焦 **UI 现代化** —— 底栏玻璃 + 切换动画、签到日历重画，以及两个用户实测的体感 bug。

## What's New

- **底部导航换玻璃 + 切换动画** — 学 iOS 26 / Telegram：tab 切换时图标 outline → filled 带一点回弹，文字字重 + 颜色平滑过渡，按下时整 tab 缩 0.92，搭配 haptic 选择反馈。底栏不再是不透明色板，而是真模糊穿透下方页面内容（BackdropFilter），Android / Windows 也一并升级，跨平台统一观感。
- **签到日历重画** — 之前的 ✅⭐⛔🔴⏳ emoji 在三星 / 小米 / 苹果机型上长得三种样，整张日历观感不一致。改成色彩弱底纹（已签翡翠绿 / 补签琥珀）+ 角标小图标 + 漏签红点 + 今日描边圈。月份切换换成紧凑 chevron 圆角按钮，月份字号放大。Legend 同步用色块 + 图标对应。
- **Android 磁贴图标重画** — 旧图标双环加中间一块小杆子，远看像手铐。按 launcher logo 比例重新画过：去掉那块杆子，两环间距收到品牌一致的 28% 重叠，cherry-stem V 接到环交点上。

## Bug Fixes

- **切换节点 UI 卡在旧节点 / 有时 spinner 一直转** — 用户实测：点新节点后流量实际已切到新节点（mihomo 已生效），但 App 还显示旧节点（比如 "台湾"），偶尔 NodeTile 上的小转圈也永远不停。根因是切换成功后内部去拿最新状态那一步偶尔会失败被静默吞掉，本地选中态没刷新。现在改成切换成功后立即更新本地选中态，后台异步对账，spinner 也保证一定会停。
- **Android Quick Settings 磁贴图标看着像手铐** — 见上 What's New。

## Upgrade Notes

- 已登录用户无需重登，订阅与设置自动保留。
- AltStore / SideStore / TrollStore 重装新 IPA 即可。
- macOS / Windows / Linux：直接覆盖。
- Android：直接安装新版 APK。
- 玻璃底栏让页面 body 延伸到栏后做模糊；Dashboard 等长列表底部最后一两个 item 会被玻璃带轻微覆盖（同 iOS Music / TG 行为），略往上滑即可看全。
