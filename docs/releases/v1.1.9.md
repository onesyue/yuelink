## Improvements

- **我的页流量数字升一档** — 当前用量 / 总量数字字号从 13pt 升到 15pt（与首页核心信息字号一致），并启用等宽数字，刷新时数字位不再抖动。
- **引导页 / 媒体页标题 token 化** — 引导步骤标题、首次使用提问、悦视频媒体详情和库标题统一收敛到 `pageTitle` token，字号、字重、行高跨页面一致。
- **右对齐写法统一** — 全工程显式右对齐统一为 `TextAlign.end`，与桌面端国际化标准对齐。

## Upgrade Notes

- 已登录用户无需重登，订阅与设置自动保留。
- 默认匿名遥测仍保持关闭；开启位置：设置 → 隐私 → 匿名遥测。
- 直接覆盖安装即可，所有平台。
